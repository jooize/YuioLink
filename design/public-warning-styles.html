YuioLink

Wieldy Ephemeral Link

Link Type
Convenient link with 1–3 words. Not secret! Private link with 4 words (in 47-bit namespace). Single-use link with 4 words. Security
Public names are short words from a public wordlist, so anyone can run the whole list and turn up every public link. Ideal for convenience and easy sharing — never for anything secret. Browse the wordlist →
Link name is generated within a 47-bit namespace — about 153 trillion possibilities — and its destination or content stays on the server only until the link expires. Even a botnet guessing a million names a second could check only ~0.4% of that space within the 7-day maximum lifetime.
Deleted from the server when revealed, and with the same security as Private links. Recipients first open the link, then have a choice to reveal its contents, which is when the one time is spent.
